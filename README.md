# admin-gestion
This is template dashboard in vue 2 and codeigniter 4.

# Installation
Clone the repository.
```
$ git clone https://github.com/Arouna170/admin-gestion.git
```
Access the repository and install dependances .
```
$ cd admin-gestion
$ npm install
```
Start codeigniter server.
```
$ php spark serve
```
